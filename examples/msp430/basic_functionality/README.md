# Basic Usage Example

This example demonstrates the core functionality of the library in a minimal environment. It shows how to perform basic operations and verify that the library is correctly integrated into your project.

## How to Build
To compile this example, use `Makefile`. You should say the `Makefile` where your `msp430-gcc` is located. For example:

```bash
GCC_PATH=/opt/ti/msp430-gcc make
```
## How to flash
To flash this example to your MSP430G2553 board you can use:
```bash
make flash
```
